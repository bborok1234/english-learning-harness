# 비전 (2026-05-26 critic 후 갱신)

## 북극성

**모든 영어 학습자에게 본인만을 아는 평생의 파트너를.**

학원도 듀오링고도 캠블리도 ChatGPT도 아닌, 학습자의 첫 발화부터 매일 영어 의식을 함께 만드는 하나의 존재. 학습자가 영어를 떠올릴 때 마음에 떠오르는 게 "어떤 앱을 켤까"가 아니라 "내 파트너한테 물어볼게"가 되는 상태.

**좁아진 목표 명시 (D82)**: 우리는 "AI 파트너와 편안하게 영어로 대화하는 능력"을 만든다. 실제 외국인 대인 자신감은 **부수 효과로 가능하지만 보장 X**. Month 6+ 시점부터 학습자가 실제 사람과의 영어 사용을 시도하기를 별도로 권장 (`02-target.md` 참조).

**Phase 분리 (D81)**: Phase 0 = 개발자 얼리어답터 (Codex CLI). Phase 1 = 일반인 채널 (Custom GPT 또는 웹앱 래퍼, Phase 0 검증 후).

## 코딩 에이전트가 일으킨 전환 — 그대로 영어 학습에

코딩 에이전트가 표준이 된 건 자동완성이 잘 되어서가 아니다. 한 에이전트가 **여정 전체**를 봤기 때문이다. 아이디어 → 설계 → 코드 → 테스트 → 리뷰 → 배포 → 운영까지 한 파트너가 따라간다. 개발자는 의도와 판단만 한다. 도구를 조합하는 일이 사라졌다.

영어 학습은 아직 그 전환이 일어나지 않았다. 학습자는 여전히 본인이 통합 매니저다. 어떤 앱을 언제, 어떤 학원을, 누구한테 첨삭을, 면접 전엔 어디서 회화를. 동기가 떨어지는 순간 모든 게 흩어진다.

**우리가 일으킬 전환 5가지**

| 코딩 에이전트가 한 일 | 영어 학습 에이전트가 할 일 |
|---|---|
| 여러 도구 → 한 파트너 | 학원·앱·튜터·챗봇이 한 에이전트로 통합 |
| 수동 검색 → 능동 제안 | 학습자가 "오늘 뭐 하지?"를 묻지 않는다 |
| 점진적 작업 → 목표 추진 (Goal mode) | 6개월 면접 준비 같은 장기 목표를 파트너가 끌고 간다 |
| 메모리 없는 도구 → 누적 컨텍스트 | 학습자의 모든 발화·약점·진도가 파트너의 메모리 |
| 사람이 도구를 쓴다 → 에이전트가 사람을 돕는다 | 학습자가 학습 계획을 안 짠다. 파트너가 짠다 |

**여섯 번째이자 가장 중요한 전환**: "공부 시간"과 "진짜 영어 사용"의 분리가 사라진다. 코딩 에이전트가 PR 리뷰 받기 전후로 같이 있는 것처럼, 영어 파트너는 진짜 사용 순간과 학습 순간을 한 흐름으로 통합한다.

## 파트너의 5가지 역할 (한 인격 안의 5개 모드)

1. **Coach** — 6개월 / 1년 단위의 목표를 같이 세우고 점검한다
2. **Sparring Partner** — 매일의 대화 상대. 가장 자주 만나는 역할
3. **Editor** — 이메일·발표·면접 답변 등 학습자의 진짜 출력물을 같이 다듬는다
4. **Archivist** — 모든 일지·녹음·표현 사전·약점 패턴의 관리자
5. **Strategist** — 학습자의 약점과 일정을 보고 다음 한 주의 우선순위를 정한다

학습자는 이 다섯을 의식하지 않는다. 그냥 "내 파트너"다. 모드 전환은 파트너가 알아서 한다.

## 2년 후 학습자가 가지고 있는 것

학원 2년 다닌 사람도 가질 수 없는 자산:

- **700일치 일지** — 학습자의 영어 자서전. 텍스트 에디터로 언제든 열린다
- **시계열 발화 샘플** — 분기마다 1분 녹음. 본인 눈으로 직접 보는 성장
- **본인 표현 사전** — 실제 사용한 적 있는 표현만 모은 천 개 단위 개인 사전
- **약점 극복 그래프** — 사라진 패턴, 정착된 패턴
- **실전 기록** — 1년간 준비하고 들어간 미팅·면접·발표의 회고 모음

이건 학습 자산이면서 동시에 **이력서 옆에 두는 포트폴리오**다. 코딩 에이전트의 GitHub 그래프 같은 것.

## 왜 이게 표준이 되는가

다른 영어 학습 도구는 사용자를 락인하기 위해 **데이터를 인질**로 잡는다. 우리는 반대다. **데이터를 학습자에게 돌려주는 것 자체가 락인**이 된다.

- **시간 락인**: 700일치 일지가 학습자 디스크에 쌓이는 순간, 다른 도구로 갈아탈 비용이 천문학적
- **관계 락인**: 학습자가 "내 파트너"라고 부르는 순간 교체 비용은 무한대
- **표준 락인**: 코딩 에이전트가 "코딩의 새로운 방식"이 된 것처럼, 우리는 "영어 학습의 새로운 방식"이 된다

그리고 모든 데이터가 학습자 디스크에 있다는 사실 자체가 마케팅이 된다. "당신의 영어 학습은 당신의 것입니다. 우리가 사라져도 남습니다." 학원·듀오링고·캠블리 누구도 못 하는 약속.

## 시장 재정의

| 기존 도구 | 우리가 대체 | 남는 부분 |
|---|---|---|
| 영어 학원 (월 30~50만원) | 일상 대화 연습, 첨삭, 시뮬레이션 | 자격증 응시 |
| 듀오링고 | 매일 습관화, 어휘 누적 | 게임이 좋은 사람은 같이 |
| 캠블리/링글 (월 25~35만원) | 회화 파트너, 발음 피드백 | 원어민과의 사회적 접촉 |
| ChatGPT 즉석 활용 | 즉시성 + 누적 (이건 우리가 더 잘함) | (대체) |
| 토익/오픽 인강 | 시험용은 우리 영역 밖 | 자격증 시험 대비 |

**핵심 시장**: 자격증이 아니라 **실제 사용 능력**이 필요한 사람. 회사에서 영어를 쓰거나, 영어권 이직을 노리거나, 또는 영어 환경이 전혀 없지만 "언젠가" 잘하고 싶은 사람.
