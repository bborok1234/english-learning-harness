# 통합 커리큘럼 — 1년 학습 여정과 모든 결정의 종합

design 단계의 capstone. 10개 영역을 한 학습 여정으로 엮는다. 타겟 페르소나 4명 + 비-타겟 인접 1명 참고 시나리오, 전체 80개 결정 인덱스, 구현 진입 체크리스트를 종합한다.

확정일: 2026-05-26

---

## 1. 좁아진 북극성 재확인

> **"AI 파트너와 편안하게 영어로 대화하는 능력."**

모든 결정은 "이게 AI 파트너와의 영어 대화 능력을 더하는가?"로 검증한다. 실제 외국인 앞 자신감은 부수 효과로 가능하지만 보장하지 않는다. 시험 / 학술 / Native 발음 / 비즈니스 문서는 우리 영역 밖.

---

## 2. 학습 여정 마스터 타임라인

### 시간축 큰 그림 (페르소나 평균값)

| 구간 | 주요 활동 | 핵심 측정 | 동기 메커니즘 |
|---|---|---|---|
| **Day 0** (onboarding) | 5분 인터뷰 + baseline 1문장 | baseline 저장 | Autonomy (시간/모드 선택) |
| **Week 1** (Day 1~7) | Free conversation 위주 | 안정화 | Identity 시드 ("매일 만나는 사람") |
| **Week 2** (Day 8~14) | + Shadowing / Picture / Read-aloud | Day 14 mini-mirror | Newness 보충 (Dropout zone 1) |
| **Week 3** (Day 15~21) | + Roleplay / Retell / Vocab stretching | 활동 다양화 | Flow (challenge-skill 균형) |
| **Week 4** (Day 22~30) | + Comparing / 4/3/2 | Day 30 monthly mirror | Competence (baseline 비교) |
| **Month 2** (Day 31~60) | 활동 자동 흐름 | 패턴 자각 | Identity 강화 |
| **Month 3** (Day 61~90) | Goal mode 활성화 (학습자 선택) | 분기 baseline | Flow + Goal |
| **Month 4-6** | 자동화 진입 (procedural → automatic) | speaking rate 향상 | Competence |
| **Month 7-12** | 학습자 시나리오 본격 | 진짜 변화 | Identity + Relatedness |

### 학습자가 한 해 후 갖고 있는 것 (재확인)

- 365일치 일지 (text)
- 분기 baseline 4개 (audio + text)
- 자주 쓰는 표현 1,000+ (vocab_seen.json)
- 약점 극복 그래프 (patterns.json)
- 본인 발음 변화 (선택)
- "내 파트너"와의 관계

## 3. 페르소나 4명 × 1년 여정 (재훈은 비-타겟 인접 — D84)

페르소나 D 재훈은 2026-05-26 비-타겟 인접으로 재분류 (`13-learner-typology.md`, `02-target.md`). 페르소나 검증 게이트(D50)는 아래 4명만 적용. 재훈 시나리오는 참고용 기록.



### A. 지은 (32, 한국 기업, Ideal-driven Introvert)
- **Day 0**: baseline "I like coffee". 출근길 8시 약속
- **Week 1**: Coffee 토픽 매일 → 어휘 누적
- **Week 2**: Shadowing 도입 (sentence-level), Picture description
- **Day 14 mini-mirror**: "I like X" 5번, "It's a X morning" 3번 — 본인 패턴 첫 자각
- **Month 2**: 자신감 살짝 — Day 30 baseline 듣고 차이 느낌
- **Month 3**: Goal mode — "AI 파트너 앞에서 영어로 피하지 않기" 정체성 강화
- **Month 6**: 실제 사람과의 영어 사용은 권장 시점. 시도 여부와 결과는 보장하지 않음
- **Month 12**: AI 파트너와의 영어 의식 정착. 365일 일지에 자기 변화 자각

### B. 민호 (42, Trauma-driven, High Anxiety)
- **Day 0**: baseline 매우 망설임 (10초 멈춤). 점심시간 약속
- **Week 1**: Read-aloud + Listening 위주. 영어 발화 부담 0
- **Week 2**: Shadowing (sentence-level) — 안전한 출력. Roleplay 안 도입
- **Day 14**: 분량 살짝 늘기 시작. trauma 단어 (미팅/본사) 여전히 회피
- **Month 2**: 회사 영어 시나리오 점진 도입 (가벼운 인사부터)
- **Month 3**: Day 60 "변한 게 없어요" 표현 → 17번 진보 정체 3단계 처리
- **Month 6**: 본사 미팅 시뮬레이션 1회 (안전 환경) → trauma 약화
- **Month 12**: 실제 본사 미팅 1회 — 완벽 X, 도망 X = 성공

### C. 수진 (28, Extrovert Identity-driven)
- **Day 0**: baseline 길게 ("I work as a designer..."). 저녁 8시 약속
- **Week 1**: Free conversation 빠르게 — 디자인 토픽
- **Week 2**: Storytelling, Picture description 빠르게 도입
- **Day 14**: 컨퍼런스 시뮬레이션 첫 등장. 회화 자신감 빠르게
- **Month 2**: Goal mode 일찍 ("1년 뒤 디자인 컨퍼런스"). Roleplay 본격
- **Month 3**: 4/3/2 도입. 자동화 부스터
- **Month 6**: 트위터 영어 활동 시작 (외부 사용). 우리는 보조
- **Month 12**: 실제 컨퍼런스 영어 발표 또는 외국 디자인 커뮤니티 참여

### D. 재훈 (38, Ought-to + Introvert — 가장 위험)
- **Day 0**: onboarding에서 정직 안내 ("시험은 학원이 더 잘해요"). 새벽 6시 약속
- **Week 1**: 거의 못 옴 (3/7일 출석). 안전망 강화. autonomy 극대화 메시지
- **Week 2**: Listening + Read-aloud 만. Output 강요 0
- **Day 14**: 학습자가 "그만둘까?" 표현 → mini-mirror 부드럽게 + 자율 강조
- **Month 2**: Ought-to → Ideal 전환 시도. "내가 원하는 거 뭐예요?" 인터뷰
- **Month 3**: 전환 성공 시 정체성 굳어짐. 실패 시 자연스럽게 끊김 (우리 OK)
- **Month 6+**: 일부 학습자만 살아남음. 정착하면 천천히 진보
- **Month 12**: 정착했다면 — 회사 압박 없이도 매일 영어 의식 가짐

### E. 혜원 (35, 영어 전공 출신, Reactivation)
- **Day 0**: baseline 길게 ("It's been a while..."). 오전 10시 30분 약속
- **Week 1**: 자가 라벨 ("다 까먹었어요") 받지 않음. 잠재력 빠르게 확인
- **Week 2**: 모든 활동 빠르게 도입. Shadowing으로 자동화 회복
- **Day 14**: 본인이 놀람 — "어 내가 이걸 알고 있었네". Identity 회복
- **Month 2**: 가족 시나리오 (아이용 영어, 가족 여행). Goal mode 일찍
- **Month 6**: 아이 영어 친구 부모와 small talk. 일상 사용 시작
- **Month 12**: 해외 이주 가능성 평가 또는 가족 여행 영어로 운영
