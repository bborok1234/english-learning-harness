# 타겟 페르소나

## 핵심 타겟: "영어 환경에 노출되지 않은 사람"

다른 영어 학습 도구의 가정 — "사용자는 어느 정도 영어를 쓸 상황이 있다" — 을 깬다. 우리는 정반대 가정에서 출발한다.

## 이 사람은 누구인가

- 회사에서 영어 거의 안 쓴다. 외국인 동료 없다
- 일상에 영어 트리거가 없다 (해외 여행도 가끔)
- 학원 한두 번 끊었다가 한 달 만에 안 갔다
- 전화영어 시작했다가 강사 바뀌면서 흐지부지
- 듀오링고 200일 했는데 외국인 만나면 한 마디도 못 한다
- 본인 영어 수준 모른다 (또는 과소/과대 평가)
- 동기가 추상적이다 ("언젠가 잘했으면 좋겠다")
- 영어 한 마디 하는 것 자체가 부담이다

## 이 사람의 진짜 두려움

- "내가 영어 못한다는 게 노출되는 순간"
- "또 작심삼일이 될까봐"
- "이번에도 돈만 내고 안 가게 될까봐"
- "내 영어가 부끄럽다"
- "외국인 만나면 도망친다"

## 이 사람의 추상적 동기

- "외국인 만나도 안 도망가고 싶다"
- "여행 가서 안 막히고 싶다"
- "영상 자막 없이 보고 싶다"
- "이력서에 영어 한 줄이라도"
- "언젠가 외국에서 살아보고 싶다"
- "그냥 매일 조금씩 익숙해지고 싶다"

## 우리가 대체하는 것

이 타겟에게 우리는 **회화학원 + 전화영어**의 자리를 차지한다. 토익/오픽 학원도 아니고 듀오링고도 아니다. **1:1 대화 상대 + 외부 강제력**이 두 가지가 핵심이다.

| 차원 | 회화학원 | 전화영어 | 우리 |
|---|---|---|---|
| 비용 | 30~50만원/월 | 25~35만원/월 | ChatGPT 구독 안 |
| 시간 강제력 | 출석 강제 (결석시 죄책감) | 통화 강제 (결강시 죄책감) | 알림만. 놓쳐도 OK |
| 첫날 부담 | 레벨테스트 + 그룹 자기소개 | 강사가 영어로 인사 | 한국어 5분, 영어 1문장 |
| 매번 시작점 | 강사 바뀌면 자기소개 다시 | 강사가 매번 같은 small talk | 어제 한 말에서 이어짐 |
| 학습 기록 | 강사 메모 (학습자 못 봄) | 강사 메모 | 학습자 디스크 마크다운 |
| 동기 의존 | 강사 카리스마 | 강사 친절도 | 누적되는 본인 자산 |

## 타겟이 우리를 만났을 때 가져야 할 첫 인상

다섯 가지가 5분 안에 일어나야 한다 (자세한 onboarding 설계는 `04-onboarding.md`):

1. 부담이 0이 되는 첫 경험
2. "이건 다르다"는 인식
3. 본인 영어로 한 마디 한 안전한 경험
4. 다음에 다시 만날 약속
5. 작은 성공 — "5분 안에 뭔가 했다"

이게 5분 안에 일어나면 학습자는 다음날 다시 온다. 안 일어나면 안 온다. 그게 전부다.

## 좁아진 목표 (북극성 보강 — 2026-05-26 critic 후 갱신)

**우리가 만드는 것**: **AI 파트너와 편안하게 영어로 대화하는 능력**.

전문적 시험이나 고차원 언어 학습이 아니다. 학습자가 다음을 갖게 되는 게 목표:
1. AI 파트너와의 영어 대화에 부담 없음 (저(低)불안 발화 환경)
2. 의미가 통하고 자연스럽게 들리는 회화 능력
3. 매일 5분 영어 의식의 정체성 형성

**가정 (미검증)** — D82로 닫음:
위 능력이 실제 대인 자신감으로 전이될 수 있다. 그러나 우리는 이것을 **보장하지 않는다**. AI 단독 학습이 대인 자신감으로 전이된다는 학술 실증은 부족하며 (우리가 인용한 AI chatbot WTC 연구들은 모두 교실 보조 도구 맥락), Month 6+ 시점에 학습자가 실제 사람과 영어 대화를 시도하는 것 (Tandem / HelloTalk 같은 언어 교환 앱)을 별도로 권장한다.

다음은 명시적으로 **우리 범위 밖**:
- 토익/오픽/IELTS 같은 시험 영어
- 학술 영작 (논문, 비즈니스 문서)
- 영문법 완벽성
- Native 수준의 발음
- 동시통역 / 고급 어휘 습득
- 영어 교육자 양성
- **실제 외국인 대인 자신감의 보장** — 부수 효과 가능, 보장 X

이 좁힘이 모든 design 결정의 잣대다. "이게 AI 파트너와의 영어 대화 능력을 더하는가?"로 매번 검증한다.

## 플랫폼-타겟 Phase 분리 (D81로 닫음)

C1 (플랫폼-타겟 불일치) 해소:

### Phase 0 — 개발자 얼리어답터
- Codex CLI 설치 가능한 기술 친화적 직장인
- 또는 도와줄 사람이 있는 학습자
- Primary 페르소나: **수진 (28, 디자이너 스타트업)**
- 페르소나 지은/민호/재훈/혜원은 Phase 0 도달이 어려움 → Phase 1로 미룸
- Phase 0 목표: 5~20명 얼리어답터 + Sol 인격 검증 + 학습 효과 입증

### Phase 1 — 일반인 채널 (v1, Phase 0 검증 후)
- ChatGPT Custom GPT 또는 웹앱 래퍼로 비개발자 진입 장벽 제거
- Primary 페르소나: **지은 (32, 한국 기업 회사원)** + 민호/혜원
- Phase 1에서 본격적인 비개발자 확장
- Phase 0의 design 자산을 그대로 활용 (skills / hooks / agents.md → web 어댑터)

## 비-타겟 인접 학습자 (D84, 2026-05-26 critic M2)

페르소나 D 재훈 같은 학습자는 우리 적합 X. 페르소나 검증 게이트(D50)는 **4명만 (지은/민호/수진/혜원)**:

- Ought-to-driven (회사 압박, 자격증 요구)
- 학원 4회+ 끊김 (외재 동기 패턴)
- 시험 영어 우선 (토익/오픽)

이런 학습자가 onboarding 진입 시 정직한 self-screen:

> "시험은 학원이 더 잘해요. 우리는 회화 쪽이에요. 그래도 회화 자신감 원하면 — 매일 5분, 부담 없이. 안 맞으면 언제든 닫아도 OK."

진입하면 환영하되, dropout을 매니페스토 위반으로 보지 않음. 페르소나 검증 게이트에서 제외.

## 비-타겟 (지금은 안 노린다)

- 이미 영어를 잘 쓰는 직장인 (재미로 더 늘리고 싶은 사람) — Phase 2 이후
- 토익/오픽/IELTS 점수가 필요한 사람 — 우리 영역 밖
- 학술 영작/비즈니스 문서 작성이 목표인 사람 — 우리 영역 밖
- 외국에서 거주 중인 사람 — 환경이 이미 있다
- 영어 교육자 — 다른 제품
- 학생 (중고등학생) — 시험 영어가 우선이라 핏 안 맞음

## 페르소나 카드: 지은 (Primary)

- 32세, 회사원, 한국 기업 근무
- 영어 마지막 사용: 3년 전 회사 메일 한 번
- 과거 시도: 학원 2회 (각 한 달), 전화영어 1회 (2주), 듀오링고 200일 (포기)
- 동기: "외국인 만나도 안 도망가고 싶다"
- 두려움: "또 작심삼일이 될까봐"
- 가능 시간: 출근길 5분 (오전 8시)
- 평가: 본인 영어를 "초급"이라고 자기 라벨링했지만 실제론 중급 진입 직전 (라벨링이 발화를 막는 케이스)

지은이 한 달 후 "내일 또 와야지"라고 생각하면 우리가 성공한 것이다.
