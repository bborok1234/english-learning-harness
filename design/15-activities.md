# 학습 활동 다양화 — 14가지 활동 카탈로그

좁아진 목표 (회화 자신감) 안에서 효과적인 활동을 정리하고, SLA 5단계 + 페르소나 5명에 매핑한다. 의존성 순서 다섯 번째 영역 (D).

확정일: 2026-05-26

---

## 1. 우리 MVP의 14가지 활동

좁아진 목표 + 일상 5분 안에서 의미 있는 활동들. **이 14개가 우리 도구가 지원할 활동 카탈로그**:

| # | 활동 | 핵심 메커니즘 | 일일 5분 적합도 |
|---|---|---|---|
| 1 | **Free conversation** | 잡담. Krashen input + Swain output 통합 | ✅ 디폴트 |
| 2 | **Roleplay** | 시나리오 기반 역할극. agency + 감정 참여 | ✅ |
| 3 | **Shadowing** | 즉시 따라하기. **2025 systematic review로 효과 입증** | ✅ |
| 4 | **Retell** | 짧은 입력 → 본인 말로 재진술. comprehension + output 통합 | ✅ |
| 5 | **Read-aloud** | 소리내어 읽기. 안전한 출력 (즉흥 부담 0) | ✅ 진입용 |
| 6 | **Picture description** | 이미지 묘사. 어휘 + 구조화 | ✅ |
| 7 | **Storytelling** | 짧은 이야기 만들기. 창의 + 구조 | 🟡 (Inter+) |
| 8 | **Vocabulary stretching** | 자주 쓰는 단어 → 한 단계 위 동의어 (i+1) | ✅ |
| 9 | **Comparing** | 같은 의도 두 가지 표현 비교 | ✅ |
| 10 | **Continuous speaking 4/3/2** | 같은 주제 4분 → 3분 → 2분. 자동화 부스터 | ❌ 5분엔 부적합, weekly milestone |
| 11 | **Tongue twisters** | 발음 놀이. 부담 0 + 자신감 | ✅ |
| 12 | **Pronunciation drill** | 음소 반복 — l/r, f/v, th, 강세 박자 | ✅ |
| 13 | **Listening (passive)** | 짧은 입력 청취. silent period / 진입용 | ✅ 진입용 |
| 14 | **Dictation** | 받아쓰기. 청해 + 어휘 + 철자 | 🟡 |

**제외한 활동들 (좁아진 목표 안에서 부적합)**:
- ❌ Grammar in context — 직접 문법 강조는 매니페스토 위반
- ❌ Translation (한↔영) — 논란 있음, 한국 학습자 L1 transfer 강화 위험
- ❌ Debate — Advanced 영역, 우리 영역 밖

## 3. 페르소나 × 활동 매핑

각 페르소나에게 어떤 활동이 효과적인지 (단계 + Personality + Anxiety 고려):

### A. 지은 (Early/Emergence, Introvert, Medium Anxiety)
**주력**: Free conversation (가벼운 잡담), Picture description, Vocabulary stretching, Shadowing (선택적)
**피해야**: Roleplay (Week 3까지), Continuous 4/3/2 (Week 4까지)

### B. 민호 (Emergence/Inter, Trauma-driven, High Anxiety)
**주력**: Read-aloud (안전한 출력), Shadowing (sentence-level), Listening
**점진적 도입**: Roleplay Week 4+ (회사 시나리오는 Week 8+)
**피해야**: 본사 미팅 시나리오 첫 한 달, Pronunciation drill (자기 평가 트리거)

### C. 수진 (Intermediate, Extrovert, Low Anxiety)
**주력**: Free conversation, Roleplay, Retell, Comparing, Continuous 4/3/2 (Week 3+)
**도입 빠름**: Storytelling Week 2부터, Picture description Week 1부터
**피해야**: 너무 단순한 활동 (지루함)

### D. 재훈 (Early Production, Ought-to, Medium-High Anxiety)
**주력**: Listening, Read-aloud (대본 의지), Pronunciation drill (단순한 것), Tongue twisters (부담 0)
**점진적 도입**: Free conversation Week 3+
**피해야**: 모든 즉흥성 활동 첫 2주, Roleplay 첫 한 달

### E. 혜원 (Intermediate, Reactivation)
**주력**: Free conversation (감 회복), Shadowing (자동화 회복), Vocabulary stretching, Storytelling
**도입 빠름**: 모든 활동 Week 2부터 가능
**피해야**: Pronunciation drill 단독 (잠재력 깎는 신호)

## 4. 활동 자동 제안 로직

모델이 매 세션 시작에 어떤 활동을 제안할지 결정하는 알고리즘:

```
input:
  - stage (백그라운드 추정, 14-stage-theory)
  - personality + anxiety (profile.md typology, 13-learner-typology)
  - recent_7_activities (interleaving 위해)
  - learner_day_count (Week 1~5 로드맵)
  - learner_condition (오늘 컨디션, optional)

logic:
  1. 학습자 단계 + Personality에 적합한 활동 풀 추출 (위 매트릭스)
  2. 최근 7일에 자주 한 활동 제외 (interleaving — Week 2+)
  3. 학습자 컨디션 신호 (피곤 → 안전한 활동 / 적극 → 도전 활동)
  4. 한국어 한 줄 옵션 제시: "Quick chat or shadowing today?"
  5. 학습자 명시 거부 가능. 디폴트 = Free conversation

output:
  - 오늘의 활동 1개 (5분 안에 한 가지)
  - 다음 세션 추천 1-2개 (학습자에게 안 보임, profile에 메모)
```

**핵심 원칙**:
- **5분 안에 한 활동만** (집중 효과)
- **거부 가능** (자율성 = 매니페스토)
- **백그라운드 학습** (학습자가 선택을 의식 못 해도 모델은 학습)

## 5. Week 1~5 활동 도입 로드맵

페르소나 평균값으로 본 도입 일정 (개별 학습자는 자기 페이스):

| Week | 활동 추가 | 이유 |
|---|---|---|
| Week 1 (Day 1~7) | Free conversation 위주 | 안정화. 활동 다양화는 Day 7 mini-mirror 이후 |
| Week 2 (Day 8~14) | + Shadowing (1-2회), Picture description (1-2회), Read-aloud (가끔) | Interleaving 시작 (`09-decisions.md` D39) |
| Week 3 (Day 15~21) | + Roleplay (1-2회), Retell (1-2회), Vocabulary stretching | Deliberate practice 본격 (D40) |
| Week 4 (Day 22~30) | + Comparing, Continuous 4/3/2 (weekly milestone) | 자동화 부스터 |
| Week 5+ | 학습자 단계 + 약점 + 시나리오에 따라 자동 흐름 | — |

**예외**: Pre-production 학습자는 Week 1에 Listening + Read-aloud 우선. Free conversation은 자연스러운 emerge 후.

## 6. 4/3/2 기법 — 우리의 Weekly Milestone 활동

**4/3/2 효과** (검증된 기법):
- 같은 주제를 4분, 3분, 2분으로 반복 발화
- 시간이 줄면서 **자동화** 일어남
- Speed + breakdown fluency 동시 향상

**우리의 적용**:
- 매주 1회 (주말 Mirror 세션의 일부)
- Week 3부터 도입 (Intermediate 단계 진입한 학습자만)
- 학습자 선택 가능 — 명시 거부 OK
- 주제는 학습자가 정함 (관심사 반영)
- 끝나면 본인 발화 3번 들으며 차이 자각 — 자체 mirror

## 7. Codex 환경에서 활동 구현 (간략)

각 활동이 우리 Codex 기반 하네스에서 어떻게 작동:

| 활동 | 구현 |
|---|---|
| Free conversation | 텍스트 / 음성 자유. 모델 응답은 학습자 단계 +1 |
| Roleplay | 모델이 시나리오 + 캐릭터 설정. 음성 가능 (CLI realtime) |
| Shadowing | 모델 TTS로 짧은 문장 → 학습자 STT로 즉시 따라하기 → 3-5회 반복 |
| Retell | 모델 짧은 텍스트/오디오 → 학습자 본인 말로 재진술 |
| Read-aloud | 학습자에게 단계 +1 텍스트 제시 → 음성으로 읽기 |
| Picture description | `gpt-image-2`로 학습자 관심사 이미지 생성 → 묘사 |
| Storytelling | 키워드 3개 → 학습자가 짧은 이야기 |
| Vocabulary stretching | 학습자 자주 쓰는 단어 → 한 단계 위 1-2개 변형 (mirror 통합) |
| Comparing | 학습자 한 표현 → 모델이 자연 변형 1개 → 학습자가 선택 |
| Continuous 4/3/2 | 모델이 시간 측정 + 학습자 발화 3회 녹음 → 비교 재생 |
| Tongue twisters | 모델이 단계 적합 twister 제시 → 학습자 따라 → TTS 모델 발음 비교 |
| Pronunciation drill | 학습자 약한 음소 (l/r, f/v, th) 단어 5개 → 반복 |
| Listening | 모델 TTS로 짧은 문장 → 학습자 yes/no 또는 한 단어 답 |
| Dictation | 모델 TTS로 짧은 문장 → 학습자 텍스트 입력 |

**기술 의존성**: 음성 활동 (3, 5, 11, 12, 13, 14)은 Codex realtime voice 모드 필요 (`06-voice-ux.md` 참조).
