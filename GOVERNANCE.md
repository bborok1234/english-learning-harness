# Governance

English Learning Harness is currently maintained by the repository owner.

## Decision Rules

- Learner safety and privacy come before feature scope.
- Local-first setup must remain the default.
- Claims about learning outcomes, realtime voice, public install, or public distribution require smoke tests or documented evidence.
- Breaking changes should include migration notes or a clear reset path.

## Release Direction

The intended public distribution surface is this source repository as an open-source project. Artifact publication remains optional and secondary.
